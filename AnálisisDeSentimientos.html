Análisis de sentimientos - Python

Análisis de sentimientos de texto, utilizando diferentes librerías en Python, que permiten la extracción y procesamiento de datos obtenidos de fuentes como las redes sociales o búsquedas en internet, el obejtivo es entender el texto y procesarlo, para generar conclusiones acerca de las emociones y sentimientos expresados.

Proyectos

Análisis de sentimientos de Twitter respecto a las votaciones presidenciales Colombia 2022

En este proyecto se realiza un ejercicio básico para efectuar la extracción de datos de un tema de interés en Twitter, procesar los textos de 5000 tweets y así establecer cuál es la polaridad de cada uno de ellos, además de la generación de una nube de términos qué permita establecer las temáticas tratadas.

Web Scraping o extracción de datos de Amazon

Por medio de BeautifulSoup se realiza la extracción de información de artículos, precios, calificaciones y opiniones de la web de Amazon. Con estos datos se desarrolla un análisis de sentimientos para establecer la polaridad de las opiniones de los productos, así como la generación de una base de datos.
