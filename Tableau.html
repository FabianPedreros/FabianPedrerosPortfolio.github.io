Dashboards - Visualización en Tableau

Proyectos de visualización de datos, ya sea para ejecutar análisis exploratorio de datos o para crear dashboards interactivos. Los dashboards creados pueden ser de tipo estratégico, operativo o de análisis, todos estos tienen la facilidad de convertir datos fríos y complejos, en visualizaciones que facilitan el entendimiento y análisis de los mismos.

Proyectos

Dashboard - Data Set Pingüinos Palmer

Dashboard interactivo creado en Tableau para facilitar el análisis de las características físicas de los pingüinos del archipiélago Palmer. Las gráficas de funciones de densidad o histogramas permiten establecer algunas diferencias en el comportamiento estadístico de atributos cualitativos y cuantitativos entre las tres razas de pingüinos; Adelié, Gentoo y Chinstrap.

  • Tableau Public

AtliQ Sales Dashboard

Dashboard operativo en el que se realiza por medio de una historia, el análisis de datos históricos de ventas de una empresa india de tecnología llamada AtliQ. Llegando a algunas conclusiones interesantes acerca de los motivos por los cuales las ventas han disminuido los últimos años y la necesidad de ampliar el portafolio de clientes.

  • Tableau Public

Covid 19 Dashboard

Dashboard created with information of cases, deaths and vaccinations in the world since the beginning of the Covid19 pandemic, this data has been previously managed in SQL Server. These five design concepts are used: 1. Gather Requirements 2. Create a template 3. Use icons and art to your advantage 4. Choose colors that matter 5. Fonts.
